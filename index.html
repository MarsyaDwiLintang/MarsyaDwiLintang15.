Dart menggunakan mixin untuk multiple inheritance tidak seperti Java/C#, Drt bisa gabungkan beberapa kemampuan digunakan untuk menghindari duplikasi kode. Contoh kasus: Bebek bisa berenang & terbang, Smartphone kamera & telepon

01. Studi Kasus Burung dan Bebek

Objek bisa punya lebih dari satu kemampuan, misalnya: Burung terbang Bebek terbang + berenang

}

    print('Can swim!');
  }

class Bird with CanFly {}

class Duck extends Bird with CanSwim {}

void main() {

  bird.fly();

  Duck duck = Duck();
  duck.fly();
  duck.swim();
}

02. Studi Kasus Mixin dengan State

Mixin juga bisa memiliki state atau kondsi seperti status terbang dan berenang, sehingga tidak hanya menambahkan fungsi tetapi juga menyimpan kondisi object.

DART SOURCE CODE

// Nama  : Marsya Dwi Lintang
// NIM   : 251410028
// Kelas : SI2A

mixin CanFly {
  bool _isFlying = false;
  bool get isFlying => _isFlying;

  void fly() {
    if (!_isFlying) {
      _isFlying = true;
      print('Started flying!');
    } else {
      print('Already flying!');
    }
  }

  void land() {
    if (_isFlying) {
      _isFlying = false;
      print('Landed!');
    } else {
      print('Already on the ground!');
    }
  }
}

mixin CanSwim {
  bool _isSwimming = false;
  bool get isSwimming => _isSwimming;

  void swim() {
    if (!_isSwimming) {
      _isSwimming = true;
      print('Started swimming!');
    } else {
      print('Already swimming!');
    }
  }

  void stopSwim() {
    if (_isSwimming) {
      _isSwimming = false;
      print('Stop swimming!');
    } else {
      print('Already on stop swimming!');
    }
  }
}

class Bird with CanFly {}
class Duck extends Bird with CanSwim {}
class Fish with CanSwim {}
class Airplane with CanFly {}

void main() {
  Bird bird = Bird();
  bird.fly();
  bird.land();

  Duck duck = Duck();
  duck.fly();
  duck.swim();

  Fish fish = Fish();
  fish.swim();

  Airplane airplane = Airplane();
  airplane.fly();
}

Pengantar Class Modifiers

Class Modifiers yang digunakan untuk mengontrol bagaimana calss digunakan, terutama dalam hal inheritance dan keamanan struktur kode, seperti sealed, base, dan final.

03. Sealed Class

Sealed Class hanya bisa diwariskan dalam file yang sama sehingga lebih aman dan semua subclass dapat dikontrol.

DART SOURCE CODE

// Nama  : Marsya Dwi Lintang
// NIM   : 251410028
// Kelas : SI2A

sealed class ApiResponse {}

class Success extends ApiResponse {
  final String message;
  Success(this.message);
}

class Error extends ApiResponse {
  final String errorMessage;
  Error(this.errorMessage);
}

void main() {
  // contoh penggunaan sederhana
  ApiResponse res1 = Success("Berhasil");
  ApiResponse res2 = Error("Gagal");

  print(res1);
  print(res2);
}

04. Keuntungan Sealed Class

Sealed Class memudahkan pengecekan kondisi karena semua kemungkinan subclass sudah diketahui, sehingga lebih aman saat digunkan dengan if atau switch.

DART SOURCE CODE

// Nama  : Marsya Dwi Lintang
// NIM   : 251410028
// Kelas : SI2A

sealed class ApiResponse {}

class Success extends ApiResponse {
  final String message;
  Success(this.message);
}

class Error extends ApiResponse {
  final String errorMessage;
  Error(this.errorMessage);
}

void handleResponse(ApiResponse response) {
  if (response is Success) {
    print("Success: ${response.message}");
  } else if (response is Error) {
    print("Error: ${response.errorMessage}");
  }
}

void main() {
  handleResponse(Success("Data berhasil"));
  handleResponse(Error("Terjadi kesalahan"));
}

05. Base Class

Base Class digunakan untuk membatasi inheritance, di mana calss hanya boleh diwariskan dengan extends sehingga struktur tetap terkontrol.

DART SOURCE CODE

// Nama  : Marsya Dwi Lintang
// NIM   : 251410028
// Kelas : SI2A

base class Animal {
  void speak() {
    print("Hewan mengeluarkan suara");
  }
}

base class Dog extends Animal {
  @override
  void speak() {
    print("Anjing menggonggong ");
  }
}

void main() {
  Animal animal = Animal();
  animal.speak();

  Animal dog = Dog();
  dog.speak();
}

07. Final Class

Final Class tidak bisa diwariskan sehingga hanya digunakan langsung untuk objek.

DART SOURCE CODE

// Nama  : Marsya Dwi Lintang
// NIM   : 251410028
// Kelas : SI2A

final class Settings {
  final String theme;
  final int fontSize;

  Settings({required this.theme, required this.fontSize});

  void showSettings() {
    print("Theme: ${theme}, Font Size: ${fontSize}");
  }
}

void main() {
  Settings settings = Settings(theme: "Dark", fontSize: 14);
  settings.showSettings();
}

Perbandingan Class Modifiers

  • Sealed Class, digunakan untuk membatasi subclass hanya dalam satu file, sehigga semua turunan calss bisa dikontrol dan lebih aman.
  • Base Class, digunakan sebagai calss dasar yang masih boleh diwariskan, tetapi tetap dengan aturan tertentu agar struktru tetap terjaga.
  • Final Class, tidak bisa diwarisakan sama sekali, sehingga hanya bisa digunakan langsung tanpa adanya turunan.

08. Praktik 1 Sealed Class dengan Pattern Matching

Slide ini menggunakan fungsi handleStatus untuk menangani berbagai kondisi dari Status, lalu dipanggil di dalam main().

DART SOURCE CODE

// Nama  : Marsya Dwi Lintang
// NIM   : 251410028
// Kelas : SI2A

sealed class Status {}

class Loading extends Status {}

class Success extends Status {
  String data;
  Success(this.data);
}

class Error extends Status {
  String message;
  Error(this.message);
}

void handleStatus(Status status) {
  switch (status) {
    case Loading():
      print("App sedang loading...");
    case Success(:var data):
      print("User: $data");
    case Error(:var message):
      print("Gagal: $message");
  }
}

void main() {
  handleStatus(Loading());
  handleStatus(Success("Login berhasil"));
  handleStatus(Error("Koneksi gagal"));
}

09. Studi Kasus Sistem State Management

Menggunakan sealed class status untuk merepresentasikan kondisi aplikasi seperti loading, success, dan eror. Setiap kondisi ditangani menggunakan fungsi handleStatus dengan switch (pattern matching) agar program dapat merespons sesuai keadaan.

DART SOURCE CODE

//Nama : Marsya Dwi Lintang
//NIM  : 251410028
//Kelas: SI2A

sealed class AppState {}

class LoadingState extends AppState {}

class SuccessState extends AppState {
  final String data;
  SuccessState(this.data);
}

class ErrorState extends AppState {
  final String message;
  ErrorState(this.message);
}

void handleState(AppState state) {
  switch (state) {
    case LoadingState():
      print("Loading...");
    case SuccessState(:var data):
      print("Success: $data");
    case ErrorState(:var message):
      print("Error: $message");
  }
}

void main() {
  handleState(LoadingState());
  handleState(SuccessState("Data berhasil"));
  handleState(ErrorState("Terjadi error"));
}

Ringkasan & Best Practices

Menggunakan sealed class AppState untuk merepresentasikan kondisi aplikasi seperti loading, success, dan error. Setiap kondisi diproses menggunakan switch di handleState, sehingga program dapat merespons sesuai keadaan.

  • Gunakan sealed class untuk memastikan semua kemungkinan state sudah terdefinisi.
  • Gunakan nama class yang jelas seperti LoadingState, SuccessState, dan ErrorState.
  • Gunakan switch (pattern matching) agar penanganan state lebih rapi dan aman.
  • Pisahkan logika (handleState) dari main() agar kode lebih terstruktur. Gunakan final pada variabel untuk menjaga data tidak berubah.

10. Bonus Base Class Repository

Menunjukkan konsep inheritance, di mana class ProductRepository mewarisi dan mengubah method dari Repository sesuai kebutuhan.

DART SOURCE CODE

//Nama : Marsya Dwi Lintang
//NIM  : 251410028
//Kelas: SI2A

class Repository {
  void create(String data) {
    print("Menambahkan data: $data");
  }

  void read() {
    print("Menampilkan semua data");
  }

  void update(String data) {
    print("Mengubah data menjadi: $data");
  }

  void delete(String data) {
    print("Menghapus data: $data");
  }
}

class ProductRepository extends Repository {
  @override
  void create(String data) {
    print("Produk berhasil ditambahkan: $data");
  }

  @override
  void read() {
    print("Daftar semua produk ditampilkan");
  }

  @override
  void update(String data) {
    print("Produk berhasil diperbarui: $data");
  }

  @override
  void delete(String data) {
    print("Produk berhasil dihapus: $data");
  }
}

void main() {
  var repo = ProductRepository();

  repo.create("Laptop");
  repo.read();
  repo.update("Laptop Gaming");
  repo.delete("Laptop");
}

11. Bonus Final Class Mathutils.

Penggunaan utility class Mathutils dengan method static untuk operasi matematika tanpa perlu membuat object.

DART SOURCE CODE

//Nama : Marsya Dwi Lintang
//NIM  : 251410028
//Kelas: SI2A

class MathUtils {
  static int tambah(int a, int b) => a + b;

  static int kurang(int a, int b) => a - b;

  static int kali(int a, int b) => a * b;

  static double bagi(int a, int b) {
    if (b == 0) {
      print("Error: tidak bisa dibagi 0");
      return 0;
    }
    return a / b;
  }
}

void main() {
  print("Tambah: ${MathUtils.tambah(8, 2)}");
  print("Kurang: ${MathUtils.kurang(8, 2)}");
  print("Kali: ${MathUtils.kali(8, 2)}");
  print("Bagi: ${MathUtils.bagi(8, 2)}");
}
